# MoviesMagazine

simple App for listing movies using MVVM with Arch components mady by Java and Kotlin.

the project uses:

1-Java and Kotlin.

2-MVVM.

3-Dagger 2 with dependency injection (including dagger android).

4-RxJava2.

5-View Model (Arch Component).

6-Live Data (Arch Component).

7-Room (Arch Component).

8-Data binding.
